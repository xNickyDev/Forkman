# -help

## Usage

```
-help {page}
```

## Arguments

| Name | Description             | Type   | Required |
| ---- | ----------------------- | ------ | -------- |
| page | The number of the page. | Number | No       |

## Prefixes

```
-
@Forkman
```

{% hint style="info" %}
The prefix cannot be changed with any command. Only the 2 stated prefixes above are working for Forkman.
{% endhint %}
