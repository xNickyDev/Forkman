# /battlepass challenges

Returns all current battle pass challenges.

## Usage

```
/battlepass challenges {page}
```

## Arguments

| Name | Description             | Type   | Required |
| :--: | :---------------------: | :----: | :------: |
| page | The number of the page. | Number | No       |
