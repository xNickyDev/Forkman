# /battlepass info

Returns information about a battle pass season.

## Usage

```
/battlepass info {season}
```

## Arguments

| Name   | Description                      | Type   | Required |
| :----: | :------------------------------: | :----: | :------: |
| season | The season to return info about. | Number | No       |
