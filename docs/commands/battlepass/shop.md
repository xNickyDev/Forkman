# /battlepass shop

Returns all current battle pass shop rewards.

## Usage

```
/battlepass shop {page}
```

## Arguments

| Name | Description             | Type   | Required |
| :--: | :---------------------: | :----: | :------: |
| page | The number of the page. | Number | No       |
