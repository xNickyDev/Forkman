# /changelogs

Returns all changelogs of Wolvesville.

## Usage

```
/changelogs {page}
```

## Arguments

| Name | Description             | Type   | Required |
| :--: | :---------------------: | :----: | :------: |
| page | The number of the page. | Number | No       |
