# /balance

Returns the balance of a user.

## Usage

```
/balance {user}
```

## Arguments

| Name | Description                               | Type | Required |
| :--: | :---------------------------------------: | :--: | :------: |
| user | The user whose balance is to be returned. | User | No       |
