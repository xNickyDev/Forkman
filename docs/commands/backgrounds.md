# /backgrounds

Returns all existing backgrounds.

## Usage

```
/backgrounds {page}
```

## Arguments

| Name | Description             | Type   | Required |
| :--: | :---------------------: | :----: | :------: |
| page | The number of the page. | Number | No       |
