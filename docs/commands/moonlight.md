# /moonlight

Returns all existing moonlight membership offers.

## Usage

```
/moonlight {page}
```

## Arguments

| Name | Description             | Type   | Required |
| :--: | :---------------------: | :----: | :------: |
| page | The number of the page. | Number | No       |
