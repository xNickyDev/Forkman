# -clan

Returns information about the given clan.

## Usage

```
-clan <name>
```

## Arguments

| Name | Description                 | Type   | Required |
| ---- | --------------------------- | ------ | -------- |
| name | The name or ID of the clan. | String | Yes      |

> [!TIP]
> You can use the [`-id`](commands/id.md) command to get the ID of any clan.
