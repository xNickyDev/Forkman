# /bot changelog

Returns the newest changelog entries of the bot.

## Usage

```
/bot changelog {month} {year}
```

## Arguments

| Name  | Description                           | Type | Required |
| :---: | :-----------------------------------: | :--: | :------: |
| month | The month of the changelog to return. | Enum | No       |
| year  | The year of the changelog to return.  | Enum | No       |
