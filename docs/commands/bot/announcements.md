# /bot announcements

Returns the announcements of the bot.

## Usage

```
/bot announcements {page}
```

## Arguments

| Name | Description             | Type   | Required |
| :--: | :---------------------: | :----: | :------: |
| page | The number of the page. | Number | No       |
