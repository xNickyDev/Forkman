# /ranked leaderboard

Returns the leaderboard of the current ranked season.

## Usage

```
/ranked leaderboard {page}
```

## Arguments

| Name   | Description             | Type   | Required |
| :----: | :---------------------: | :----: | :------: |
| page   | The number of the page. | Number | No       |
