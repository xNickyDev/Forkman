# -leaderboard

Returns a leaderboard of the specified type.

## Usage

```
-leaderboard <type (gold/votes)> {page}
```

### Aliases

```
-lb
-top
```

## Arguments

| Name | Description                  | Type   | Required |
| :--: | :--------------------------: | :----: | :------: |
| type | The type of the leaderboard. | Enum   | Yes      |
| page | The number of the page.      | Number | No       |
