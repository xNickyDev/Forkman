# /clan message

Sends a message to the chat of your clan.

> [!NOTE]
> This command requires clan verification.

## Usage

```
/clan message <text>
```

## Arguments

| Name | Description             | Type   | Required |
| :--: | :---------------------: | :----: | :------: |
| text | The message to send.    | String | Yes      |
