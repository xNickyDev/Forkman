# /clan quest history

Returns all quests your clan has done in the past.

> [!NOTE]
> This command requires clan verification.

## Usage

```
/clan quest history {page}
```

## Arguments

| Name | Description             | Type   | Required |
| :--: | :---------------------: | :----: | :------: |
| page | The number of the page. | Number | No       |
