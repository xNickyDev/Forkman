# /clan announcements

Returns all announcements of your clan.

## Usage

```
/clan announcements {page}
```

## Arguments

| Name | Description             | Type   | Required |
| :--: | :---------------------: | :----: | :------: |
| page | The number of the page. | Number | No       |
