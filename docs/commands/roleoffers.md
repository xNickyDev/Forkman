# -roleoffers

Returns all existing advanced role card offers.

## Usage

```
-roleoffers {page}
```

## Arguments

| Name | Description             | Type   | Required |
| :--: | :---------------------: | :----: | :------: |
| page | The number of the page. | Number | No       |
