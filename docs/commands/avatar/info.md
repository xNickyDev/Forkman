# /avatar info

Returns information about a shared avatar.

## Usage

```
/avatar info <code>
```

## Arguments

| Name | Description                    | Type   | Required |
| :--: | :----------------------------: | :----: | :------: |
| code | The code of the shared avatar. | String | Yes      |
