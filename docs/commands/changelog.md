# /changelog

Returns the newest changelog entries.

## Usage

```
/changelog {month} {year}
```

## Arguments

| Name  | Description                           | Type   | Required |
| :---: | :-----------------------------------: | :----: | :------: |
| month | The month of the changelog to return. | String | No       |
| year  | The year of the changelog to return.  | String | No       |
