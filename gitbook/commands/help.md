# /help

## Usage

```
/help {page} {command}
```

## Arguments

| Name    | Description              | Type    | Required |
| :-----: | :----------------------: | :-----: | :------: |
| page    | The number of the page.  | Number  | No       |
| command | The name of the command. | Command | No       |
