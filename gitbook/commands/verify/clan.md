# /verify clan

{% hint style="info" %}
This command requires account verification.
{% endhint %}

## Usage

```
/verify clan <name>
```

## Arguments

| Name | Description                        | Type   | Required |
| :--: | :--------------------------------: | :----: | :------: |
| name | The exact name or ID of your clan. | String | Yes      |
