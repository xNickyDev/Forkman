# /vote tracker

## Usage

```
/vote tracker <action>
```

## Arguments

| Name   | Description                   | Type | Required |
| :----: | :---------------------------: | :--: | :------: |
| action | Select the action to perform. | Enum | Yes      |

### Possibilities

{% tabs %}

{% tab title="action" %}
- `Enable` - Enables the vote tracker system.
- `Disable` - Disables the vote tracker system.
{% endtab %}

{% endtabs %}
