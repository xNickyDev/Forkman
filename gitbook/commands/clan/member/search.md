# /clan member search

{% hint style="info" %}
This command requires clan verification.
{% endhint %}

## Usage

```
/clan member search <member>
```

## Arguments

| Name   | Description                                          | Type   | Required |
| :----: | :--------------------------------------------------: | :----: | :------: |
| member | The member of your clan to return information about. | String | Yes      |
