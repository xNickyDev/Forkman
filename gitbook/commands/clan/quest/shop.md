# /clan quest shop

{% hint style="info" %}
This command requires clan verification.
{% endhint %}

{% hint style="success" %}
This command allows verified **(Co-)Leaders** to buy quests for their clan.
{% endhint %}

## Usage

```
/clan quest shop
```
