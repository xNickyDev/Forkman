# /clan quest shuffle

{% hint style="info" %}
This command requires clan verification.
{% endhint %}

{% hint style="success" %}
This command allows verified **(Co-)Leaders** to shuffle quests for their clan.
{% endhint %}

## Usage

```
/clan quest shuffle
```
