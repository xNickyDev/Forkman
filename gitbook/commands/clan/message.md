# /clan message

{% hint style="info" %}
This command requires clan verification.
{% endhint %}

{% hint style="success" %}
This command allows sending messages directly to your clan chat via Discord!
{% endhint %}

## Usage

```
/clan message <text>
```

## Arguments

| Name | Description             | Type   | Required |
| :--: | :---------------------: | :----: | :------: |
| text | The message to send.    | String | Yes      |
