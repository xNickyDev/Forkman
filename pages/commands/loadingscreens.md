# /loadingscreens

Returns all existing loading screens.

## Usage

```
/loadingscreens {page}
```

## Arguments

| Name | Description             | Type   | Required |
| :--: | :---------------------: | :----: | :------: |
| page | The number of the page. | Number | No       |
